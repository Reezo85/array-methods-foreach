# Array Methods — `find` & `findIndex` Exercise (Mythical Creatures)

**Student:** Reginald Boyd  
**Goal:** Practice using `find` and `findIndex` with a creatures catalog.

## Dataset
Includes a fantasy realm of mythical creatures with type and last seen location.

## Tasks
1. Use `find` to locate the first creature of type "Water".
2. Use `findIndex` to locate the index of the "Griffin".
3. Use `find` to locate the first creature last seen in "Enchanted Forest".

## Expected Results
- First Water Creature: Mermaid
- Index of Griffin: 3
- Creature in Enchanted Forest: Unicorn
